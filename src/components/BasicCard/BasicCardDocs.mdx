# Basic Card

#
#
#

## Descrição
O BasicCard é usado na home para o usuário ter acesso as paginas da aplicação.

#
#
#

## Props

- **title**: É um string que representa o titulo da página.
- **description**: É um string que representa a descrição da página.

#
#
#

## Utilização

```
<BasicCard
  title='Formulário'
  description='Formulário simples feito com os componentes da Material UI e validações feitas com yup'
/>
```
